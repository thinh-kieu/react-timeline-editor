### Scale Customization

+ scale: Single scale mark category (>0)
+ scaleSplitCount: Number of subdivided units per scale (>0 integer)
+ scaleWidth: Display width of a single scale (>0, unit: px)
+ startLeft: Distance from the left side where the scale starts (>=0, unit: px)
