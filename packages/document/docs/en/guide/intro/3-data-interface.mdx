### TimeLineEffectSource
> Editor Running Effect Source Data Structure

+ start -- Trigger Condition
  + Triggered when the runner starts playing, if the time is within the current action time range
+ enter -- Trigger Condition
  + Entering the current action time area from a non-action time area
+ update -- Trigger Condition
  + Triggered every frame when playing the current action (including reRender)
  + Triggered during reRender
+ leave -- Trigger Condition
  + Leaving the current action time area
+ stop -- Trigger Condition
  + Triggered when the runner pauses, if the time is within the current action time range
