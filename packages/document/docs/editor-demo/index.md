# 编辑器示例

这里提供了React Timeline Editor的各种使用示例，帮助你快速了解和掌握组件的各种功能。

## 🚀 基础功能

### [基本使用](/editor-demo/editor-basic)
演示时间轴编辑器的基本功能，包括拖拽、缩放等基础操作。

### [动作配置](/editor-demo/editor-action-config)
展示如何配置动作的可移动性、可缩放性以及移动范围限制。

### [自定义样式](/editor-demo/editor-custom-style)
演示如何通过自定义渲染函数和CSS样式来定制动作块的外观。

## 🎯 交互功能

### [网格吸附](/editor-demo/editor-grid-snap)
启用网格吸附功能，让动作块按照网格单元进行精确对齐。

### [辅助线吸附](/editor-demo/editor-auxiliary-line-snap)
演示辅助线吸附功能，在拖拽时自动对齐到其他动作或光标位置。

### [自动滚动](/editor-demo/editor-auto-scroll)
当拖拽超出可视区域时，自动滚动时间轴以显示更多内容。

## 📊 高级功能

### [基础事件处理](/editor-demo/editor-basic-event)
展示如何处理动作和行的点击、双击、右键等事件。

### [移动缩放回调](/editor-demo/editor-callback)
演示如何通过回调函数控制拖拽行为，实现自定义的交互逻辑。

### [刻度自定义](/editor-demo/editor-scale-customization)
展示如何自定义时间刻度的显示方式和样式。

### [滚动同步](/editor-demo/editor-scroll-sync)
演示如何实现时间轴与其他组件的滚动同步。

## 💡 使用提示

- 每个示例都包含完整的代码实现
- 可以直接复制代码到你的项目中使用
- 建议按照从基础到高级的顺序学习
- 所有示例都支持实时预览和交互

开始探索这些示例，快速掌握React Timeline Editor的强大功能！
