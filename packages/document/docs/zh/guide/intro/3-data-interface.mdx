### TimeLineEffectSource
> 编辑器效果运行代码数据结构

+ start 触发条件
  + 运行器开始play时，如时间在当前action时间范围内触发
+ enter 触发条件
  + 从非action时间区域进入当前action时间区域
+ update 触发条件
  + 播放当前action时每帧触发（包括reRender）
  + reRender时触发
+ leave 触发条件
  + 从当前action时间区域离开
+ stop 触发条件
  + 运行器pause时，如时间在当前action时间范围内触发
